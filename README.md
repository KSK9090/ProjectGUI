# ProjectGUI

Ein Prototyp einer Benutzerschnittstelle die Ergebnisse einer verlinkten Datenbank filtern und ausgeben soll.

## Funktionen

* Option Anzahl der Ergebnisse auszugeben
* Suchfunktion die den Kontakt ausgibt

## Zukünftige Funktionen

* Einträge in die Datenbank via Entry

## Installation

* IDE der Wahl
* Python 3.x+
* Ordner entpacken und ProjectGUI.py ausführen
